ZX-Poly emulator

Author: Igor Maznitsa
Project page: https://code.google.com/p/zxpoly/
Author's page: http://www.igormaznitsa.com

It is an emulator of ZX-Poly platform. The Platform is a multi-cpu platform based on the ZX-Spectrum 128 platform. The Platform was developed by Igor Maznitsa in 1994 and doesn't contain any magic components.

The Emulator emulates below listed IO devices:

  1. Keyboard
  2. Beta-disk interface
  3. Kempston mouse (click on form - trap the mouse, ESC - free mouse)
  4. Kempston joystick
  5. Tape (only reading)
